# lalala-ui

## Project setup
```
yarn install lalala-ui
```
## to use
```
import lalalaUi from 'lalala-ui'
import "lalala-ui/lib/lalala-ui.css";
Vue.use(lalalaUi)
```
### part1
```
prop:value to control hide or show

<text-particles :value="isShow">
    <img alt="Vue logo" src="../assets/logo.png" />
</text-particles>
```
